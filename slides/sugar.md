## `LambdaCase`

::: {class="left"}
Adds syntactic sugar for pattern matching on a function's argument.
:::

##

```haskell
pretty ::
  -> Expr
  -> Text
pretty e = case e of
  LitI n -> pack $ show n
  LitB True -> "true"
  LitB False -> "false"
```

##

```haskell
pretty ::
  -> Expr
  -> Text
pretty = \case
  LitI n -> pack $ show n
  LitB True -> "true"
  LitB False -> "false"
```

## `MultiWayIf`

::: {class="left"}
Adds syntactic sugar for nested `if-then-else` expressions.
:::

##

```haskell
  if 1 < 0 then
    "foo"
  else if 12 > 4 then
    "bar"
  else if even 42 then
    "42"
  else
    "no idea"
```

##

```haskell
  if | 1 < 0 -> "foo"
     | 12 > 4 -> "bar"
     | even 42 -> "42"
     | otherwise -> "no idea"
```
